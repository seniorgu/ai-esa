# Contributing

Thank you for your interest in contributing.

This repository contains documentation, guides, and related resources. Contributions of all sizes are welcome, including:

- clarification of explanations
- suggestions and improvements
- case sharing
- new documentation pages

---

# Issues

## Please use GitHub **Issues** tab for:

- specific documentation improvements

When opening an issue, include as much detail as possible.

---

## Pull Requests

Direct contributions are always welcome through pull requests. 

---

# Contribution Guidelines

## Check Existing Issues

Before starting major work, check whether a related issue or discussion already exists.

---

## Keep Pull Requests Focused

Small, focused pull requests are easier to review and merge.

---

## Be Respectful

Please keep discussions constructive and respectful.

---

# Questions

If you are unsure about anything, feel free to open an issue or discussion.

Thank you for helping improve this project.
